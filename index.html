speedlimit

Network bandwidth limiter for testing iphone apps

Dependencies

Mac OS X 10.6 (It may still support 10.5 with 32-bit prefPane, but I haven't tested it yet)

Install

You can build this project, and just double click the release build. You can also download the 64-bit preference pane.
